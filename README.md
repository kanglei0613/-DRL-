# Deep Reinforcement Learning Experiments (Python)

## 專案簡介
本專案為大學期間深度強化學習課程之實作內容，  
目標為將強化學習理論（如 Q-learning、DQN）實際實作於模擬環境中，
並透過實驗分析模型學習行為與效能差異。

## 使用技術
- Language: Python
- Libraries: numpy, pytorch, gym
- Tools: Jupyter Notebook, Git

## 實作內容
- Q-learning / Deep Q-Network (DQN) 實作
- 強化學習環境建模與互動流程設計
- 訓練過程與結果視覺化
- 不同參數設定與學習效果比較
